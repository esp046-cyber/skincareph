# skincareph
Design Elements:
Warm Filipino-inspired color palette (beige, browns, gold tones)
1% margins throughout
Small font titles (18px max for main header)
Clean, mobile-optimized layout
No 3D elements, no search bar
Features:
4 Main Sections: Home, Ingredients, Routine, Tips
Home: Introduction to Filipino skincare basics and sun protection
Ingredients: Grid of 8 traditional Filipino skincare ingredients with icons
Routine: Step-by-step morning and evening skincare routines
Tips: Practical advice for Philippine climate and budget-friendly options
